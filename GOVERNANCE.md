# Governance

This project is governed by:

1. Ethical Governance Charter for an Open-Source eDelphi Platform
2. AI Governance & Human-in-the-Loop Contract
3. Apache License 2.0
4. Project maintainer and contribution policies

The Ethical Governance Charter is the highest project authority for ethical, methodological, and participant-protection requirements. The AI Governance & Human-in-the-Loop Contract is subordinate to the Charter and governs all AI-assisted features.

The Apache License 2.0 governs reuse of the software code. It does not override ethical, institutional, human-subjects, privacy, or study-specific governance obligations.
