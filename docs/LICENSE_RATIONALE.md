# License Rationale

The eDelphi Open Source Platform is released under the Apache License 2.0.

Apache-2.0 was selected because it is an OSI-approved permissive open-source license that supports broad public-interest adoption, auditability, institutional use, commercial and non-commercial deployment, and long-term sustainability.

The license also includes an express patent grant, which is useful for research infrastructure and public-interest software that may be adopted across universities, nonprofits, public agencies, and private organizations.

The project's ethical governance depends on transparent, auditable code. However, open-source code does not mean open study data. Participant data, identity mappings, consent records, response data, audit logs, and study exports remain private by default and are governed separately by study protocols, consent terms, institutional review, and applicable law.

Apache-2.0 governs reuse of the software code. It does not authorize misuse of the platform, deceptive claims about Delphi outputs, violation of participant confidentiality, removal of human-in-the-loop safeguards, or claims that Delphi consensus establishes correctness.
