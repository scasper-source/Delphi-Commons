# Internal Browser Rehearsal

This folder records the short internal browser rehearsal for Delphi Commons after the light baseline QC package.

Scope is limited to controlled mock-participant MVP testing with synthetic data only. This is not production validation, not IRB launch approval, and not approval for real human-subjects research or sensitive participant data.

Files:

- `REHEARSAL_SCRIPT.md` - the rehearsal script and role setup.
- `REHEARSAL_RESULTS.md` - observed results from the latest rerun, Test 4.
- `DEFECT_LOG_DELTA.md` - defects and evidence gaps found after the baseline QC.
- `GO_NO_GO_FOR_MOCK_TRIAL.md` - decision for the larger controlled synthetic mock trial.
